Designsystemet - Code Syntax

Dette er en tredjeparts plugin som kan generere CSS-syntax fra designsystemet. Den vil gå gjennom alle variablene som er tilgjengelig for designere i Figma og lage CSS-syntax som matcher den som designsystemet genererer. I fremtiden vil denne funksjonaliteten bli innebygd i designsystemets egen plugin. Men frem til da er dette en alternativ løsning.
